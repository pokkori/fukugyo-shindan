# 副業デビュー診断

会社員・副業未経験者向け。5分の診断で「自分に合った副業」と「今日から始めるチェックリスト」を表示するWebアプリ。

## 機能

- **診断**: 5問の質問に答えると、適性の高い副業を3つ表示
- **初日ステップ**: 選ばれた副業の「今日やること」をすぐ確認
- **チェックリスト**: ブラウザに進捗を保存。テンプレート（出品文・提案文など）をコピー可能
- **途中保存**: 回答を自動保存し、「続きから」で再開可能（「最初からやり直す」ボタンあり）
- **PDF出力**: 診断結果を印刷/PDFで保存可能

## 副業の種類

| 種類 | 初期資金 | 特徴 |
|------|----------|------|
| ポイントサイト | 0円 | アンケート・広告でポイント貯める |
| メルカリ | 0〜5万円 | 手持ち品 or 仕入れて出品 |
| クラウドワークス | 0円 | スキルを仕事として受注 |
| ブログ | 月1,000円前後 | 長期で資産になる |
| せどり | 5万〜10万円 | 仕入れて転売 |

## 起動

### 方法1: エージェントから起動（推奨）

日本語パスでターミナルがエラーになる場合は、**一度だけ** 以下を実行：

```cmd
C:\dev\setup-fukugyo.cmd
```

その後、エージェントは `cmd /c C:\dev\start-fukugyo.cmd`（working_directory: C:\dev）で起動可能。

### 方法2: 手動で起動

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く。

### 方法3: start.cmd をダブルクリック

プロジェクトフォルダ内の `start.cmd` をダブルクリック。

## 技術スタック

外部API・DB不要。完全無料で動作。
