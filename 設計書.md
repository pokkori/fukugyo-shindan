# 副業デビュー診断 設計書

## 1. 概要 (Overview)

会社員・副業未経験者向けの「あなたに合った副業」診断Webアプリ。5問の質問に答えると、適性の高い副業を3つ表示し、「今日から始めるチェックリスト」とテンプレートを提供する。

### 1.1 目的 (Purpose)

* 副業を始めたいが何から手をつければいいかわからない人向けに、**診断＋初日の一歩**を提供する。
* チェックリストとテンプレートで迷わず行動できるようにする。

### 1.2 対象ユーザー

* 会社勤めで副業未経験
* スキマ時間・資金・スキルに合わせた副業を探している

### 1.3 実装状況サマリ (Implementation Status)

| 領域 | 内容 | 備考 |
|------|------|------|
| 診断 | 5問のクイズ（時間・資金・スキル・興味・収益スピード） | lib/data.ts の QUIZ_QUESTIONS |
| 結果表示 | 1〜3位の副業、初日ステップ、チェックリストへのリンク | app/result/page.tsx |
| チェックリスト | 副業ごとのステップ、進捗を localStorage に保存 | app/checklist/[id]/page.tsx |
| テンプレート | 出品文・提案文などをコピー可能 | 各 SIDE_JOBS の templates |
| 共有URL | 結果ページのクエリ ids で再表示可能。「結果URLをコピーして共有」ボタンでクリップボードにコピー | ?ids=mercari,point,crowdworks |
| 結果の保存 | localStorage に保存。トップページで「前回の結果を見る」リンク表示 | STORAGE_KEY: fukugyo-last-result |

## 2. システム構成

* **フロントエンド**: Next.js 14 (App Router), TypeScript, Tailwind CSS
* **データ**: 静的データ（lib/data.ts）、進捗は localStorage
* **外部API・DB**: なし。完全無料で動作。

## 3. 画面遷移

```
/ (トップ)
  → /quiz (診断)
    → /result?ids=xxx,yyy,zzz (結果)
      → /checklist/[id] (チェックリスト)
```

## 4. 副業の種類

| ID | 名前 | 初期資金 | 特徴 |
|----|------|----------|------|
| point | ポイントサイト | 0円 | アンケート・広告でポイント貯める |
| mercari | メルカリ | 0〜5万円 | 手持ち品 or 仕入れて出品 |
| crowdworks | クラウドワークス | 0円 | スキルを仕事として受注 |
| blog | ブログ | 月1,000円前後 | 長期で資産になる |
| stock | せどり | 5万〜10万円 | 仕入れて転売 |

## 5. 診断ロジック

* 各質問の選択肢に、副業ごとのスコア（0〜3）を付与。
* 5問の合計スコアでランキング。上位3件を結果として表示。
* `lib/data.ts` の `calculateResult(answers)` で計算。

## 6. 起動

```bash
npm install
npm run dev
```

日本語パスでターミナルがエラーになる場合: `C:\dev\setup-fukugyo.cmd` を実行後、`C:\dev\start-fukugyo.cmd` で起動。
